📚 Méthodologies de cours

Retrouvez toutes les méthodologies pour réussir vos cours de CEJM, Anglais et Français.

💼

CEJM

Culture Économique, Juridique et Managériale

Syllogisme

  • Les faits : qui est qui ? Qu'est ce qui les lies ? Contexte + X se demande ...
  • Règles de droits : citer les docs + connaissances cours
  • Solutions : En l'espèce (mettre en lien la situation avec les règles de droits) En conséquences : X à le droit ou pas (ou détailler les solutions si plusieurs)

Economie - Management

  • Intro : Définitions
  • Développement : Justification de tout ce qu'il a à détaillé
  • Conclusion : Pour conclure ... (1-2 lignes max)
🇬🇧

Anglais

Langue vivante

Compréhension écrite

  • Résumé en français en 200 mots (+ ou - 10%)
  • Aucun mots anglais
  • Ce document est ... par ... publié en ..., il traite de ...
  • ATTENTION ! Pas de traduction pur du titre

Expression écrite

  • Faire des phrases complètes
  • Expression en 200 mots (+ ou - 10%)
  • Si yen a un qui veut la lettre il vient me voir

Expression orale

  • Organiser les 3 docs
  • This forlder is composed of 3 docs that deals with ... In my first part I will shed on light ... Secondly I will bring to the for ... And then ...
  • AUCUN Français !! Même pour une question ou autre
  • Se préparé un peu pour les questions
📖

Français

CGE

Questions

  • Intro : Présentations du/des docs : auteur, date, sujet + question + plan
  • Développement : Confronter les documents (très importants)
  • N'utiliser que des petites parties des docs (pas des phrases entières)
  • Petite conclusion

Essaie

  • Intro : Problématique, citer l'organisations de l'essaie
  • Développement : parties-sous parties
  • Au moins 1 référence par argument (intégrer au possible tout les docs du corpus)
  • Conclusion : Pour conlure, je pense que ... + ouverture de fin(pas obligatoire)
